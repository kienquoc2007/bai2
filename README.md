Mini Project 1 - Khóa Học HTML5 Cho Người Mới Bắt Đầu
Giới thiệu

Đây là mini project thực hành HTML5 cơ bản, được xây dựng nhằm giúp người học làm quen với:

Cấu trúc chuẩn của một trang HTML5.
Các thẻ Semantic trong HTML5.
Điều hướng nội bộ bằng thẻ <nav>.
Hiển thị hình ảnh, bảng dữ liệu và biểu mẫu (Form).
Tổ chức nội dung theo các khu vực rõ ràng.
Cấu trúc trang web

Trang web gồm các thành phần chính:

Header
Hiển thị tên khóa học.
Banner giới thiệu nội dung học tập.
Navigation

Menu điều hướng đến các phần:

Tổng Quan Bài Học
Nội Dung Chi Tiết
Bài Tập Thực Hành
Hỗ Trợ & Góp Ý
Main Content
I. Tổng Quan Bài Học
Giới thiệu về Semantic Tags trong HTML5.
Hiển thị hình ảnh minh họa giao diện.
II. Nội Dung Chi Tiết
Bảng tổng hợp các nhóm thẻ HTML5 quan trọng.
Giải thích chức năng của từng nhóm thẻ.
III. Nộp Bài Tập Thực Hành

Biểu mẫu cho phép học viên:

Nhập họ tên.
Nhập liên kết GitHub Repository.
Ghi chú bổ sung.
Gửi bài lên hệ thống.
Footer
Thông tin tác giả.
Năm phát hành dự án.
Công nghệ sử dụng
HTML5

Không sử dụng CSS hoặc JavaScript trong phiên bản hiện tại.

Mục tiêu học tập

Sau khi hoàn thành project này, học viên có thể:

Hiểu cấu trúc cơ bản của một trang HTML5.
Sử dụng các thẻ Semantic đúng mục đích.
Tạo bảng dữ liệu bằng <table>.
Tạo biểu mẫu bằng <form>.
Tổ chức nội dung website một cách khoa học và dễ bảo trì.
